• ngStyle: Sets CSS style on an HTML element conditionally.
  • ngClass: Sets CSS classes on HTML an element, dynamically.
  • Warning: Do not use class names with - characters!
<select ng-model="buttonSize" ng-options="size for size in [ 'small', 'normal', 'big' ]"> <option value="">-- Choose Size --</option> </select> <button ng-class="{ btn: true, btnSmall: buttonSize == 'small', btnBig: buttonSize == 'big'}">Button</button> <span ng-repeat="colorValue in ['red', 'blue', 'green']" ng-style="{ color: colorValue, 'font-size': '15px' }"> {{colorValue}} </span>
