• Angular has created directives for each main form element, with custom attributes and behavior.
<form name="userForm" ng-init="user = { login: null, mail: 'Bob@gmail.com', role: 'user' }"> Login: <input type="text" ng-model="user.login" ng-minlength="5" ng-maxlength="10" /> Email: <input type="email" ng-model="user.mail" ng-required="true" /> Account type: <input type="checkbox" ng-model="user.role" ng-true-value="admin" ng-false-value="user"> {{currentUser.role}} </form>
