• Protractor is the E2E framework test for Angular
  • Built on WebdriverJS aka Selenium Server
  • Replace ngScenario, the initial way to perform E2E test
  • Introduced at AngularJS v1.2 release presentation
  • Allows running tests targeting remote addresses
  • Has it's own runner, no need for Karma
  • Can run tests on browsers in parallels (thanks to selenium grid)

Getting started

> npm install protractor --save-dev > ./node_modules/protractor/bin/webdriver-manager update > ./node_modules/protractor/bin/webdriver-manager start
  • The update may fail behind proxy for now. In that case please download :
    • The Selenium server at : selenium-server-standalone-2.39.0.jar
    • A Browser Driver at : chromedriver_win32.zip
