• End-to-end tests are placed in the test/e2e/ directory of your application.
  • Each scenario is composed of one or more of these blocks:
    • it: Made of commands and expectations.
    • beforeEach: Run before each it block.
    • afterEach: Run after each it block.
  • ngScenario provides severals commands to manipulate and test the application and its rendered HTML.
  • You can use class and id selectors (. and #) if JQuery is loaded in your tests.
