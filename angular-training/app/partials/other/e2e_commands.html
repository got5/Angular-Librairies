• element(selector).click(): Clicks on a given element.
  • element(selector).text()/.html(): Returns the content of a given element.
  • input(ng-model).enter(value): Enters the value in the text field binding to the scope property ng-model.
  • input(ng-model).val(): Returns the current value of an input field.
  • pause(): Pauses the execution of the tests.
  • sleep(seconds): Pauses the execution of the tests for X seconds.
  • browser().navigateTo(url): Loads the url.
  • browser().location().path(): Returns current path.
  • expect(expression).{not.}{matcher}: Asserts the valud of the expression satisfies the matcher. Available matchers are: toEqual (angular.equals()), toBe (===), toMatch (compares with a given regular expression), toBeNull, toBeLessThan, etc...
