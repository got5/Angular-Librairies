• Karma is a javascript test runner, which runs on Node.js.
  • Works by opening each browser specified in its configuration file, and by running Javascript tests in these browsers.
  • Communication between Karma and each of the browsers is handled with a service running in the terminal using socket.io.
  • Very fast!!!
# Install karma # (use sudo or launch your terminal with administrator right) > sudo npm install -g karma > sudo npm install -g karma-chrome-launcher > sudo npm install -g karma-jasmine # Can also be launched with a grunt task > grunt test:unit
