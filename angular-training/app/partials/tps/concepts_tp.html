1. Get following AngularJS application skeleton.
  2. Extract the archive in your workspace and import the project in your IDE.
  3. Get and install Node.js.
  4. In your project directory, open a new console and execute these two commands:
    • npm install: Install the application and gets all its dependencies.
    • node server: Launches a server to access the application.
  5. Product description page is handled by two files: the template detail.html and the controller DetailController.js. Use the Angular two way binding to fill the template with the product informations.
  6. Test your application at following url: http://localhost:3000/#/book/1
