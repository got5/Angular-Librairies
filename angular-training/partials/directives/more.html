• ngDisabled: Makes the button enabled/disabled.
  • ngReadOnly: Makes the input read only.
  • ngInclude: Fetches, compiles and includes an external HTML fragment.
  • ngBind: Replaces the text content of the specified HTML element with the value of a given expression. Same as {{expression}}. Useful to hide bindings while the page is loading.
  • ngCloak: Prevents the HTML template from being briefly displayed by the browser in its raw/uncompiled form while your application is loading.
  • ngBindHtmlUnsafe: Creates a binding that will innerHTML the expression result.
  • ngHref: Used instead of href.
