• Package manager for client-side components.
  • Search/install packages: bower search underscore/bower install underscore.
  • Can be used with AngularJS to find, install, update and publish reusable components.
# Search for a component by name bower search angular Search results: - angular git://github.com/angular/bower-angular.git - angular-resource git://github.com/angular/bower-angular-resource.git ... # Search options bower search angular | grep "cookies" ... # Download/install a component in a components directory and add it to bower.json bower install component-name --save # Download a component with a specific version. bower install component-name#version # Register you component and make it appear in searches bower register [app name] [git endpoint]
