1. We will now handle the login functionality in our ecommerce application. Files to edit are the template login.html and its controller LoginController.js.
  2. Inject the $http service to load the file /data/users.json.
  3. If the $http service returns an error when trying to load the json file, use the $log service to log an error in the console.
  4. Check if the login/password entered by the user refer to an existing user in the json file. If so, use the service $location to redirect the user to the main page.
  5. When the user is successfully logged, use the $cookies service to save the user login.
