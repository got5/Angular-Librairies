• Protractor is the E2E framework test for Angular
  • Built on WebdriverJS aka Selenium Server
  • Allows running tests targeting remote addresses
  • Has it's own runner, no need for Karma

Getting started

> npm install protractor > node_modules/protractor/bin/webdriver-manager update > cp -r node_modules/protractor/selenium/* config/scripts/
  • The update may fail behind proxy for now. In that case please download :
    • The Selenium server at : selenium-server-standalone-2.40.0.jar
    • A Browser Driver at : chromedriver_win32.zip
