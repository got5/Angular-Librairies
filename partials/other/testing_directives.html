• Follow the steps below before testing your directives:
    1. Load the template of the directive.
    2. Create an HTML element which uses your directive.
    3. Create a new scope, and provide any data needed by your directive.
    4. Use the service $compile to compile your html element with the newly created scope.
    5. You can now use the generated html to test the behavior of your directive.
describe('starRating directive unit test', function() { var elem, scope; // Load the directive. beforeEach(module('app')); // Load the directive template. beforeEach(module('templates/partials/starRating.html')); // Done before each test. Creates the tested DOM. beforeEach(inject(function($rootScope, $compile) { // Create a html snippet which uses the directive to test. elem = angular.element('<star-rating></star-rating>'); scope = $rootScope; scope.rating = 3; // Compile the HTML snippet. $compile(elem)(scope); scope.$digest(); })); // TODO: tests, using elem and scope. });
